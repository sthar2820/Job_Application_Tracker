# Sample Email Data

This directory is for storing anonymized sample emails for testing purposes.

**⚠️ IMPORTANT: Never commit real emails containing personal information!**

## Structure

Place anonymized `.txt` files here with sample email content for testing agent parsing.

Example filenames:
- `confirmation_greenhouse.txt`
- `rejection_lever.txt`
- `interview_workday.txt`
- `assessment_hackerrank.txt`
- `offer_generic.txt`

## Example Format

```
Subject: Your Application to Acme Corp - Software Engineer
From: jobs@greenhouse.io
Date: 2024-01-15

Dear Candidate,

Thank you for applying to the Software Engineer position at Acme Corp.
We have received your application and will review it shortly.

Best regards,
Acme Recruiting Team
```

## Privacy

All emails in this directory should be:
- Anonymized (no real names, emails, phone numbers)
- Generic company names (Acme Corp, Example Inc, etc.)
- Sanitized of any identifying information
